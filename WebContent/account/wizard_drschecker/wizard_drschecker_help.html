Clipc DRS Checker

This tool validates a given resource for the CLIPC DRS Metadata standard.

It suggests the dataset DRS and filename DRS according to the global DRS attributes in the NetCDF file.

It is possible to make direct links of NetCDF Opendap URLS to this page by using the resource= key value pair:
  • https://climate4impact.eu/impactportal/account/wizard_drschecker.jsp?resource=<the opendap file>

For example:
  • https://climate4impact.eu/impactportal/account/wizard_drschecker.jsp?resource=http://opendap.knmi.nl/knmi/thredds/dodsC/CLIPC/drs_tests/gsi_nco-4-4-8_CERFACS_multi-platform-tier2v1_day_1989-2010.nc

See https://dev.knmi.nl/projects/drschecker/wiki for more info and how to use this tool as a commandline script.
